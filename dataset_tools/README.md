# Dataset tools
Scripts used for dataset post-processing


### 1. find_multi_stable_poses.py
A script that selects objects with at least n stable poses by examining their "stable_poses.npy" files.
### 2. add_stable_pose_to_metadata.py
A script that, based on the TXT file generated by find_multi_stable_poses.py, marks objects in metadata.csv that have at least n stable poses.
